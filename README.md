# Galoy Status

This repository hosts a trial Galoy status page powered by [Upptime](https://upptime.js.org).

Upptime uses:

- GitHub Actions for uptime checks
- GitHub Issues for incidents
- GitHub Pages for the public status page

The monitored endpoints are configured in [`.upptimerc.yml`](./.upptimerc.yml).

After the first Upptime workflow run, the generated status page should be available at:

https://galoymoney.github.io/galoy-status/
